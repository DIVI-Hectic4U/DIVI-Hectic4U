### 🎮 HackYourFriend — Real-Time 1v1 Multiplayer Typing Game
> `Unity` `C#` `Netcode for GameObjects` `Unity Relay`

**Solo-architected** a real-time multiplayer typing game supporting **20+ concurrent players** at a sustained **100–150ms P2P latency** via Unity Relay. Engineered a deterministic state machine and custom input parsing engine across **10+ C# scripts**, synchronizing keystrokes and executing **2 live sabotage mechanics** (backspace lock, key-scrambling) through RPCs. Leveraged Network Variables for smooth, latency-compensated game state synchronization.
